# Build and Test

Compile the extension and verify it works.

## Steps

1. **Install dependencies** (if needed):
   ```bash
   npm install
   ```

// turbo
2. **Build the extension:**
   ```bash
   make build
   ```

3. **Run lint** (when configured):
   ```bash
   make lint
   ```

4. **Test manually:** Press `F5` in VS Code to launch the Extension Development Host and verify the terminal button appears.
